## Como compilar
No terminal, digite: 
```bash
$ make
```
O makefile está configurado para compilar com todas as flags de warnings e ao final gera um arquivo executável no diretório /bin.

## Como rodar
No terminal, digite:
```bash
$ ./bin/prog3
```
